Configuration Files¶

There are two types of configuration file for saving and reusing common properties for OLIGOSS sequencing workflows. Both of these configuration files are in JSON format:

  1. Instrument Configuration<Instrument-Config>

  2. OLIG Files<OLIG-Files>
